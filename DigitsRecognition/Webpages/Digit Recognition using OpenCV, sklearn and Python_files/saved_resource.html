                                        Hi Marek,

                                        I wrote this code using OpenCV 2.4.8 and I am assuming that you are using OpenCV 3. If that is the case, the cv2.findContours function has changed and hence you are getting the error.

                                        ctrs, hier = cv2.findContours(im_th.copy(), cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

                                        _, ctrs, hier = cv2.findContours(im_th.copy(), cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
